Some of my Recent Projects

Pack Scheduler

Pack Scheduler was one of my favorite projects that I completed in my time at NC State because I was able to make sense of something I had to do every semester and know what was actually going on behind the scenes! This project mocks the process of registering for classes that all NC State students do with each new semester. Our version of this course registration system provides users with all of the functionality that the normal system used by NC State provides. The program has dashboards for users dependent on their role: student, teacher, or registrar. Each role has different functionalities and views for these users. Overall, this project taught me many of the principles of Object Oriented Programing. This program demonstrates the use of object oriented programming with abstraction, while also using encryption, data structures, and IO.
